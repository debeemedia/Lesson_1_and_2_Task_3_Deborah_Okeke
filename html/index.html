How to create an HTML file in VSCode

Creating an html file in vscode is very easy. We will be looking at four steps to create an html file in Visual Studio Code.

Step 1: Launching VSCode

Go to the Windows Start Menu and click on the VS Code icon to launch it.

Step 2: Creating the File

Click on new file and a dialog box will pop up for you to type the name of the file and choose a file location.

Step 3: Adding the File Extension

Ensure that after typing the file name, you add .html at the end
